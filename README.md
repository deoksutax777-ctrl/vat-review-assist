# 부가세 신고서 결재 보조

매입매출장 엑셀(위하고·세무사랑)을 브라우저에서 파싱해 유형별 집계·신고서 간이 대사를 수행하고, Claude Sonnet API로 매입세액 공제·불공제 의심 항목을 스크리닝하는 단일 HTML 도구.

**사용**: https://deoksutax777-ctrl.github.io/vat-review-assist/

1. 우측 상단 ⚙에서 본인의 Anthropic API 키 등록 (브라우저 localStorage에만 저장, 서버 전송 없음)
2. 매입매출장 xlsx 드래그&드롭 → 유형별 집계 확인
3. 신고서 주요 항목 수기 입력 → 간이 대사
4. [Sonnet 스크리닝 실행] → 공제·불공제 의심 항목 플래그

## 데이터 처리

- 엑셀 파싱은 전부 브라우저 안에서 처리되며 어떤 서버로도 업로드되지 않음
- AI 스크리닝 실행 시에만 매입 요약(거래처×계정과목×유형 집계)이 본인 API 키로 Anthropic에 직접 전송됨 (거래처명 마스킹 옵션 제공)
- API 키를 코드에 넣지 말 것 — 화면 입력만 사용

## 고지

AI 스크리닝 결과는 확인용 참고 목록이며 세무 판단의 결론이 아님. 최종 판단은 담당 세무사의 검토로 확정한다.
